# HireHub

A web-based Employee Recruitment System that streamlines the hiring process by allowing recruiters to post jobs and candidates to apply online.

## 🚀 Features

- User Authentication
- Recruiter Dashboard
- Candidate Dashboard
- Job Posting
- Job Applications
- Database Integration
- Responsive UI

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Jinja2

## 📂 Project Structure

```
HireHub/
├── app.py
├── database.py
├── database/
├── templates/
├── static/
├── requirements.txt
└── recruitment.db
```

## ⚙️ Installation

```bash
git clone https://github.com/siddhus-droid/HireHub.git
cd HireHub
pip install -r requirements.txt
python app.py
```
